Key Design Decisions

Collapsible Sidebar
220px default, collapses to icon-only. Gives the main content area maximum space for tables and kanban.
Dark-First Design
Dark backgrounds (#09090B), subtle borders (#27272A). Light mode is equally polished. Both get equal love.
4-Column Stat Grid
Top-level KPIs always visible. Each card shows value + trend indicator. Responsive → 2-col on tablet, 1-col on mobile.
Content Hierarchy
2:1 split for main content vs secondary (activity feed). Clean, predictable rhythm. No floating cards.
